# Password Strength Checker (Python)

A simple Python password strength checker that scores passwords and provides improvement tips
using practical security rules (length, character variety, common password checks, and patterns).

## Features
- Rates passwords as **Weak / Medium / Strong**
- Scores based on:
  - Password length
  - Lowercase & uppercase letters
  - Numbers
  - Special characters
  - Common passwords (basic blacklist)
  - Repeated characters and simple sequences

## Technologies
- Python
